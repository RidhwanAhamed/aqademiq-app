# Timer Sound Files

This directory contains notification sounds for the Pomodoro Timer feature.

## Required Sound Files

The following sound files need to be added to this directory:

1. **bell.mp3** - Classic bell sound (default for focus complete)
2. **chime.mp3** - Soft chime (default for break complete)
3. **digital.mp3** - Digital beep
4. **gentle.mp3** - Gentle notification
5. **nature.mp3** - Nature sound (bird chirp)
6. **success.mp3** - Success chime

## File Requirements

- Format: MP3
- Recommended size: ~50KB or less per file
- Duration: 1-3 seconds
- Sample rate: 44.1kHz or 48kHz
- Quality: 128-192 kbps

## Usage

These sounds are played when:
- A focus session completes
- A break session completes

Users can customize which sound plays for each event type in the Timer Settings dialog.
